## 1. Как изменить реквизиты компании

### Через редактирование HTML-файла

1. Откройте файл `index.html` в текстовом редакторе
2. Найдите раздел `<section id="about" class="about section">`
3. Внутри найдите блок `<div class="requisites">`
4. Измените значения в следующих полях:

```html
<!-- Полное наименование -->
<span class="requisite-item__value">Общество с ограниченной ответственностью «Импульс»</span>

<!-- Сокращённое наименование -->
<span class="requisite-item__value">ООО «Импульс»</span>

<!-- Юридический адрес -->
<span class="requisite-item__value">101000, г. Москва, ул. Примерная, д. 10, офис 501</span>

<!-- ИНН (10 или 12 цифр) -->
<span class="requisite-item__value">7700000000</span>

<!-- КПП -->
<span class="requisite-item__value">770101001</span>

<!-- ОГРН (13 цифр) -->
<span class="requisite-item__value">1207700000000</span>

<!-- ОКВЭД -->
<span class="requisite-item__value">
    62.01 — Разработка компьютерного программного обеспечения<br>
    62.02 — Деятельность консультативная и работы в области компьютерных технологий
</span>
```

5. Также обновите реквизиты в футере (нижняя часть страницы):
```html
<p>ИНН 7700000000 | ОГРН 1207700000000</p>
```

6. Обновите Structured Data (Schema.org) в `<head>`:
```html
<script type="application/ld+json">
{
    "@type": "Organization",
    "name": "Общество с ограниченной ответственностью «Импульс»",
    "taxID": "7700000000",
    "legalName": "Общество с ограниченной ответственностью «Импульс»"
}
</script>
```

7. Сохраните файл и загрузите на хостинг

### Проверка формата данных

- **ИНН**: должно быть 10 или 12 цифр (например, `7700000000`)
- **ОГРН**: должно быть 13 цифр (например, `1207700000000`)
- **КПП**: 9 цифр (например, `770101001`)
- **ОКВЭД**: должен соответствовать IT-направлениям (см. Приложение Б в ТЗ)

## 2. Как добавить/изменить услугу и цену

### Добавление новой услуги

1. Откройте файл `index.html`
2. Найдите раздел `<section id="services" class="services section">`
3. Внутри `<div class="services__grid grid">` добавьте новый блок:

```html
<article class="service-card card">
    <div class="card__icon">🎯</div>
    <h3 class="card__title">Название услуги</h3>
    <p class="card__description">
        Описание услуги — 2-3 предложения о том, что вы делаете.
    </p>
    <div class="card__details">
        <div class="detail">
            <strong>Для кого:</strong> целевая аудитория
        </div>
        <div class="detail">
            <strong>Технологии:</strong> Python, JavaScript, PostgreSQL
        </div>
        <div class="detail">
            <strong>Стоимость:</strong> от 100 000 руб.
        </div>
        <div class="detail">
            <strong>Сроки:</strong> от 2 недель
        </div>
    </div>
</article>
```

### Изменение стоимости услуги

1. Найдите раздел `<section id="prices" class="prices section">`
2. В таблице `<table class="prices__table">` измените значения в ячейках:

```html
<tr>
    <td>Название услуги</td>
    <td>Описание услуги</td>
    <td>от 100 000 руб.</td>
    <td>от 2 недель</td>
</tr>
```

3. Также обновите цену в карточке услуги (см. выше)

### Требования к описанию услуг

- Все тексты должны быть **на русском языке**
- Обязательно указать:
  - Что входит в услугу
  - Для кого предназначена
  - Используемые технологии (с переводом или пояснением)
  - Стоимость в рублях (или «по запросу»)
  - Сроки выполнения

## 3. Как обновить контактную информацию

### Обновление контактов в нескольких местах

1. **Раздел Контакты** (`<section id="contacts">`):

```html
<!-- Адрес -->
<p>101000, г. Москва, ул. Примерная, д. 10, офис 501</p>

<!-- Телефон -->
<p><a href="tel:+74950000000">+7 (495) 000-00-00</a></p>

<!-- Email -->
<p><a href="mailto:info@impuls-it.ru">info@impuls-it.ru</a></p>

<!-- Режим работы -->
<p>Понедельник — Пятница: 9:00 — 18:00 (МСК)</p>
```

2. **Футер** (`<footer class="footer">`):

```html
<div class="footer__column">
    <h4>Контакты</h4>
    <ul>
        <li><a href="tel:+74950000000">+7 (495) 000-00-00</a></li>
        <li><a href="mailto:info@impuls-it.ru">info@impuls-it.ru</a></li>
        <li>г. Москва, ул. Примерная, д. 10</li>
    </ul>
</div>
```

3. **Structured Data** (в `<head>`):

```html
<script type="application/ld+json">
{
    "contactPoint": {
        "telephone": "+7-495-000-00-00",
        "email": "info@impuls-it.ru"
    },
    "address": {
        "streetAddress": "ул. Примерная, д. 10, офис 501",
        "addressLocality": "Москва",
        "postalCode": "101000"
    }
}
</script>
```

4. **JavaScript** (файл `js/main.js`):

```javascript
const requisitesData = {
    email: 'info@impuls-it.ru',
    phone: '+7 (495) 000-00-00'
};
```

## 5. Как проверить соответствие требованиям

### Чек-лист ежемесячной проверки

Пройдитесь по следующему списку и отметьте выполненные пункты:

- [ ] Все ли тексты на русском языке?
- [ ] Есть ли переводы иностранных терминов (SaaS, CRM, ERP и т.д.)?
- [ ] Актуальны ли реквизиты компании (ИНН, ОГРН, адрес)?
- [ ] Указаны ли все обязательные ОКВЭД из IT-перечня?
- [ ] Доступен ли сайт без регистрации и паролей?
- [ ] Работают ли все формы обратной связи?
- [ ] Актуальны ли цены в прайс-листе?
- [ ] Действителен ли SSL-сертификат?
- [ ] Оплачен ли хостинг?
- [ ] Нет ли упоминаний о «сервисной компании для группы компаний ИП Ярошенко»?
- [ ] Загружаются ли все страницы не более чем за 3 секунды?
- [ ] Корректно ли отображается сайт на мобильных устройствах?

### Автоматическая проверка через консоль браузера

Откройте консоль разработчика (F12) и выполните:

```javascript
// Проверка доступности сайта
checkSiteAvailability();

// Получение текущих реквизитов
getRequisites();

// Проверка ОКВЭД на соответствие IT-перечню
validateOKVED('62.01'); // должно вернуть true
```

### Проверка через онлайн-сервисы

1. **Проверка скорости загрузки**: [PageSpeed Insights](https://pagespeed.web.dev/)
2. **Проверка SSL-сертификата**: [SSL Labs](https://www.ssllabs.com/ssltest/)
3. **Проверка доступности**: [UptimeRobot](https://uptimerobot.com/)

## 6. Что делать при обнаружении ошибок

### Если найден иностранный текст без перевода

1. Немедленно добавьте русский перевод или пояснение
2. Пример:
   - Было: «Sign up for our SaaS platform»
   - Стало: «Зарегистрируйтесь в нашей облачной платформе (SaaS)»

### Если обнаружена информация о сервисной компании для ИП Ярошенко

1. Немедленно удалите все упоминания
2. Замените на позиционирование: «Аккредитованная IT-компания»
3. Пример замены:
   - Было: «Мы — сервисная компания для группы компаний ИП Ярошенко»
   - Стало: «Мы — аккредитованная IT-компания, предоставляющая услуги по разработке ПО»

### Если не работает форма обратной связи

1. Проверьте подключение файла `js/main.js`
2. Проверьте консоль браузера на наличие ошибок
3. Убедитесь, что у формы есть `id="feedbackForm"`
4. Проверьте атрибут `action` у формы

### Если сайт недоступен

1. Проверьте статус хостинга (оплачен ли)
2. Проверьте SSL-сертификат (не истёк ли срок действия)
3. Проверьте логи сервера на наличие ошибок
4. При необходимости восстановите из резервной копии

### Если обнаружены неверные реквизиты

1. Немедленно обновите данные во всех местах (см. раздел 1)
2. Проверьте формат данных (ИНН 10/12 цифр, ОГРН 13 цифр)
3. Убедитесь, что ОКВЭД соответствуют IT-направлениям
