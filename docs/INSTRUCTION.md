## Быстрый старт

Все данные для редактирования находятся в файле `js/config.js`. Откройте этот файл в любом текстовом редакторе и измените нужные значения.

### Структура файлов:
```
/workspace/
├── index.html          # Главная страница (не требует правок)
├── css/style.css       # Стили (не требует правок)
├── js/
│   ├── config.js       # ← РЕДАКТИРОВАТЬ ЗДЕСЬ
│   └── main.js         # Логика (не требует правок)
└── docs/
    └── INSTRUCTION.md  # Эта инструкция
```

## Редактирование реквизитов компании

Откройте `js/config.js` и найдите блок `company`:

```javascript
company: {
    fullName: "Общество с ограниченной ответственностью «Магистрат»",
    shortName: "ООО «Магистрат»",
    inn: "",           // Впишите ИНН (10 или 12 цифр)
    kpp: "",           // Впишите КПП (9 цифр)
    ogrn: "",          // Впишите ОГРН (13 цифр)
    legalAddress: "",  // Впишите юридический адрес
    okved: [
        { code: "62.01", name: "Разработка компьютерного программного обеспечения" },
        { code: "62.02", name: "Деятельность консультативная и работы в области компьютерных технологий" },
        { code: "62.09", name: "Деятельность, связанная с использованием вычислительной техники и информационных технологий, прочая" },
        { code: "58.29", name: "Издание прочих программных средств" }
    ]
},
```

### Что заполнять:
- **INN** — 10 цифр для юрлиц или 12 для ИП
- **КПП** — 9 цифр (указывается в свидетельстве о постановке на учёт)
- **ОГРН** — 13 цифр
- **Юридический адрес** — полный адрес с индексом

## Изменение контактной информации

В том же файле `js/config.js` найдите блок `contacts`:

```javascript
contacts: {
    email: "info@magistrat.example",
    phone: "+7 (800) 123-45-67",
    phoneRaw: "+78001234567"
},
```

### Важно:
- `email` — отображаемый адрес почты
- `phone` — телефон в красивом формате
- `phoneRaw` — телефон без пробелов и скобок (для ссылки `tel:`)

## Обновление прайс-листа

В файле `js/config.js` найдите блок `prices`:

```javascript
prices: {
    hourStandard: {
        name: "Час работы специалиста 1С (стандартный)",
        description: "Программирование, настройка, консультации в рабочее время",
        price: 3500,
        unit: "час"
    },
    // ... другие услуги
},
```

### Как изменить цену:
1. Найдите нужную услугу по ключу (например, `hourStandard`)
2. Измените значение `price` на новое (число в рублях)
3. Сохраните файл

### Форматы цен:
- **Фиксированная цена**: `price: 3500` → отобразится как "3 500 ₽"
- **Цена за единицу**: `price: 3500, unit: "час"` → "3 500 ₽ / час"
- **По смете**: `price: null, unit: "по смете"` → "по смете"

### Список всех услуг:
| Ключ | Описание |
|------|----------|
| `hourStandard` | Час работы специалиста 1С (стандартный) |
| `hourUrgent` | Час работы специалиста (срочно / вне графика) |
| `subscription` | Абонентское сопровождение 1С |
| `implementation` | Внедрение типовой конфигурации «под ключ» |
| `update` | Обновление типовой конфигурации 1С |
| `integration` | Интеграция с внешней системой |
| `development` | Разработка отчёта / обработки / печатной формы |
| `survey` | Обследование и техническое задание на внедрение |
| `serverAdmin` | Администрирование сервера 1С (абонемент) |

## Валидация данных

Перед публикацией проверьте корректность данных:

### Проверка в браузере:
Откройте консоль разработчика (F12) и выполните:

```javascript
// Проверка ИНН
validateINN("7700000000");  // true если валиден

// Проверка ОГРН
validateOGRN("1234567890123");  // true если валиден

// Проверка КПП
validateKPP("770101001");  // true если валиден

// Проверка ОКВЭД
validateOKVED("62.01");  // true если IT-код
```

### Требования к форматам:
- **ИНН**: 10 или 12 цифр
- **ОГРН**: 13 цифр
- **КПП**: 9 цифр
- **ОКВЭД**: должен быть из IT-перечня (62.01, 62.02, 62.03, 62.09, 63.11, 58.29)

## API для программного обновления

Для интеграции с CMS доступны следующие функции:

```javascript
// Обновление реквизитов
MagistratCMS.updateRequisites({
    inn: "7700000000",
    kpp: "770101001",
    ogrn: "1234567890123",
    legalAddress: "г. Москва, ул. Примерная, д. 1"
});

// Обновление контактов
MagistratCMS.updateContacts({
    email: "new@example.com",
    phone: "+7 (999) 000-00-00"
});

// Обновление цены
MagistratCMS.updatePrice("hourStandard", 4000);

// Получение всех данных
const data = MagistratCMS.getCompanyData();

// Обновление отображения
MagistratCMS.refreshDisplay();
```

## Часто задаваемые вопросы

### Q: Как добавить новую услугу в прайс-лист?
A: Добавьте новый объект в блок `prices` в файле `config.js`:
```javascript
prices: {
    // ... существующие услуги
    newService: {
        name: "Название услуги",
        description: "Описание услуги",
        price: 50000,
        unit: ""
    }
}
```

### Q: Как скрыть реквизиты, если они ещё не заполнены?
A: Оставьте поля пустыми — вместо них отобразится «Не указан».

### Q: Можно ли изменить цвета сайта?
A: Да, в начале файла `css/style.css` есть CSS-переменные:
```css
:root {
    --color-primary: #2563eb;  /* Основной цвет */
    --color-primary-dark: #1d4ed8;
    /* ... другие переменные */
}
```
